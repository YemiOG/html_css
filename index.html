The Basic Language of the Web: HTML

Posted by Fikeyinmi Disu on Monday, June 21st 2022

All modern websites and web applications are built using three fundamental technologies: HTML, CSS, JavaScript. These are the languages of the web

In this post, let's focus on HTML. We will learn what HTML is all about, and why you should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It's a makeup language that web developers use to structure and describe the content of a webpage (it is not a programming language)

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, videos etc. Web browsers undertsand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs

There are countless reasons for laenring the fundamental language of the web. Here are 5 them:

  • To be able to use the fundamental web dev fundamental
  • The hand-craft beautiful websites instead of relying tools like Wordpress or Wix
  • To build web applications
